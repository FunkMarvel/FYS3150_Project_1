\section{Introduction} \label{sec:I}
One of the most versitile tools in modern science is numerical integration, thus it it simportant to understand its limits. In this paper we have performed numerial integration of a second order differential equation. This was done by discretizing the differential equation, and formulating it as a matrix-vector equation. The matrix-vector equation was then solved using both a general, and specialzed Thomas algorithm, as well as LU-decomposition.
